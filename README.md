# hse-project-2024
Проект 2024 года ПАДИИ - приложение-компаньон для игры в D&D. Позволяет создать игровое поле, установить карту и загрузить спрайты персонажей. 

Для запуска: склонируйте main себе на устройство, установите необходимые зависимости, запустите main. Через config можно настроить разрешение экрана (width, height)

Детали управления: двойной клик по имени персонажа откроет окно с его листом персонажа.
